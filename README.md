# 🎵 Music Lab

AI와 함께 작사/작곡을 배우고, 만들고, 출시까지 하는 프로젝트.

텔레그램 봇 [@HIH_Music_bot](https://t.me/HIH_Music_bot)으로 대화하며 음악을 학습한다.

## 기능

- **작사**: 주제/장르/분위기 → 가사 생성 + 작사 기법 설명
- **작곡**: 코드 진행 추천 + 이론 설명 + MIDI 파일 생성
- **MIDI**: 멜로디/코드 → .mid 파일 → 텔레그램 다운로드
- **이론**: 음악 이론 Q&A
- **자유 대화**: 음악 관련 아무 질문

## 사용법

```
/lyrics 비오는 날 이별 발라드
/chord 밝은 팝
/midi C장조 4마디 피아노 멜로디
/theory 마이너 스케일이 뭐야?
```

## 기술 스택

- Python 3.12 + python-telegram-bot
- Claude CLI (OAuth, API 키 불필요)
- midiutil (MIDI 파일 생성)
