## 제작 노트

### Suno 워크플로우
1. Custom Mode 사용
2. Style of Music -> 각 곡의 스타일 태그 복붙 (200자 이내)
3. Lyrics -> 가사 전체 복붙 (섹션 마커 포함)
4. Title -> 곡 제목 입력
5. 첫 생성: Verse 1 + Chorus 구간 확인 (약 2분 클립)
6. 마음에 드는 보컬/사운드 선택 -> Extend로 나머지 이어붙이기

- 인디 팝: 01, 03, 04, 06, 09, 11, 13
