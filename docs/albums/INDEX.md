## 컨셉 발굴 가이드

새 앨범/EP 영감을 정리할 때 최소 다음 6개 항목을 기록:

1. **한 줄 컨셉**: 화자가 무엇을 말하려 하는가?
2. **형식**: 싱글 / EP(6~9곡) / 앨범(9~12곡) / 인스트루멘털
3. **톤·레퍼런스**: 3~5명의 아티스트 (왜 그 아티스트인지 한 줄)
4. **가사 언어**: 한국어 / 영어 / 없음(인스트)
5. **구조 트릭**: 수미상관 / 모티프 인용 / 키 동선 / 사이클 등 — 청자가 "왜 이 순서로 들어야 하는가"를 설명하는 장치
6. **차별점**: 직전 앨범과 무엇이 다른가? (톤/가사/형식/구조)

위 6개 항목이 채워지면 `docs/albums/{제목}.md`로 상세 컨셉(트랙별 BPM/키/편성/세부) 작성 → PREPARED_TASK.md에 등록.
