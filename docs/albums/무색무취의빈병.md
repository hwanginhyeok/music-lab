# 무색무취의 빈병

> 인스트루멘털 앨범 / 9 트랙 / 초안 v0.4 (2026-05-14)

## 컨셉

향수병으로서의 자아. 본래 색도 향도 없는 빈 용기. 누군가 들어와야 향이 나지만, 그 향마저 휘발성이라 매일 조금씩 날아가고 결국 다시 무색무취로 돌아간다. 새 사람이 다가올 때마다 같은 사이클이 반복된다.

**가사 없음.** 모든 서사는 사운드·악기·다이내믹·공간감으로만 전달.

**보컬리즈는 남성, 한 명, 전 트랙 일관.** (2·6·8번 3곡에만 등장. 한 인물의 흔적이 사이클 내내 같은 목소리로 남는 효과)

**후처리 최소화 원칙**: Suno 생성 단계에서 작곡·편곡·악기 선택으로 모든 표현을 끝낸다. 별도 후처리(EQ 자동화, 멀티밴드 컴프 등) 의존 금지.

**톤 방향**: 재즈 트리오 미니멀리즘 (Bill Evans 중심) — Kind of Blue의 modal 부유감과 Chet Baker의 숨 섞인 인간성 사이.

## 핵심 모티프 (사운드 언어)

| 모티프 | 사운드 표현 |
|---|---|
| 빈병 | 공간감, 룸톤, 리버브 테일, 침묵 |
| 향이 든다 | 트리오 인터플레이 강화, 보컬리즈 진입 |
| 향이 휘발 | 악기 하나씩 빠지기, 노트 수 줄이기, 다이내믹 감소 |
| 다시 빈병 | 피아노 솔로 + 룸톤만 남음 |
| 사이클 | 1번 ↔ 9번 멜로디 수미상관, 2번 모티프 → 8번 인용 |

향수의 **탑·미들·베이스 노트** 구조를 트랙 진행에 매핑:
- 탑노트 (1-3): 만남, 화려함, 휘발성 가장 높음
- 미들노트 (4-6): 본질, 가장 오래 머무는 향
- 베이스노트 (7-9): 잔향, 사라짐, 다시 비어감

## 레퍼런스 아티스트 (3)

### Bill Evans 🇺🇸 — "음표 사이의 침묵" (중심)
1929–1980. 재즈 피아노의 시인. Debussy 화성을 재즈로 번역. 음을 거의 치지 않음으로써 가장 많이 말한다. ECM 레이블이 추구하는 "침묵을 녹음한다"의 원형. *Kind of Blue* 공동 작곡가.
- 대표곡: *Blue in Green*, *Peace Piece*, *Waltz for Debby*, *My Foolish Heart*
- 우리: **트랙 1/9 구조 교과서** (Blue in Green → 1번, Peace Piece → 9번), 수미상관 아키텍처, 트리오 내 공간 활용

### Miles Davis — *Kind of Blue* (1959) 레퍼런스
아티스트가 아닌 앨범. 재즈 역사상 가장 많이 팔린 음반. Modal jazz의 시작 — 빠른 코드 체인지 없이 한 모드 위에 부유하는 느낌. Bill Evans가 리더/공동작곡자로 참여. ppp 다이내믹의 전범.
- 트랙: *Blue in Green*, *Flamenco Sketches*, *All Blues*, *Freddie Freeloader*
- 우리: **앨범 전체의 정서적 레퍼런스**, 모달 접근법, 해소 없는 화성, 1~9번 전반의 공기감

### Chet Baker 🇺🇸 — "숨이 섞인 보이스"
1929–1988. 트럼페터 + 보컬리스트. 목소리와 트럼펫 모두 같은 방식으로 연주: 공기를 절반쯤 섞어 흘리듯. **허밍이 가사가 되고 가사가 허밍이 된다** — 경계가 없음. 우리 보컬리즈 모델.
- 대표곡: *Almost Blue*, *My Funny Valentine*, *The Thrill Is Gone*
- 우리: **2·6·8번 보컬리즈의 질감** — 숨 섞인 흘리는 허밍, 가사 없이도 인간성

## 트랙리스트 (9트랙)

> BPM·키는 v0.2 1차안. 1번/9번 만든 후 조정 가능.
> 키 동선: D minor(빈병/우울) → F major(따뜻함) → A minor(절정/단조) → C minor(체념) → D minor(회귀)

### Side A — 채워짐

#### 1. 무색무취 (Colorless, Odorless)
- **러닝타임**: 2:30
- **BPM**: 52~56 (루바토) / **키**: D minor, modal
- **편성**: 피아노 트리오 (피아노 중심, 업라이트 베이스 페달톤, 브러시 드럼 거의 무음)
- **레퍼런스**: Bill Evans *Blue in Green* (Miles Davis, Kind of Blue 1959)
- **다이내믹**: ppp
- **세부**:
  - 한 마디 3-4음. 노트 사이 침묵 2~3초.
  - 피아노 좌손: 10도 간격 open voicing, 거의 안 움직임
  - 업라이트 베이스: 워킹 아님, D 페달톤 위주, 4~6박에 한 번 움직임
  - 브러시 드럼: 공기 질감만, 거의 안 들림
  - 멜로디 모티프 A: 4음 하강형 (D-C-A-F) — 9번에서 회귀
  - 인트로 성격, 청자가 "병 안에 들어왔다" 느낌

#### 2. 시향 (First Spray)
- **러닝타임**: 3:30
- **BPM**: 72 / **키**: D minor → F major (전조)
- **편성**: 피아노 트리오 + **남성 보컬리즈 (Chet Baker 스타일)**
- **레퍼런스**: Bill Evans *Waltz for Debby* (보컬리즈 진입) + Chet Baker *My Funny Valentine*
- **다이내믹**: p → mp
- **세부**:
  - 1번 끝 룸톤이 그대로 이어짐 (트랙 단절 없음)
  - 트리오 인터플레이 점진적 강화 (8마디) → 호기심 모티프(상승 3도)
  - **보컬리즈 첫 등장** (1:30 지점). **남성, 중음역(테너~바리톤 경계, 약 G3~D4), 숨 섞인 흘리는 허밍 — Chet Baker 방식, 가사 없이 "음—" "아—" 한 음 길게**
  - 멜로디 모티프 B: 상승 5음 (F-A-C-E-G) — 8번에서 인용
  - F major 전조 = 누군가 들어옴, 따뜻해짐

#### 3. 탑노트 (Top Note)
- **러닝타임**: 4:00
- **BPM**: 96 / **키**: F major → A minor
- **편성**: 피아노 트리오 풀 인터플레이 (가장 활발한 버전 — 드럼 브러시 최대 활용)
- **레퍼런스**: Miles Davis *Freddie Freeloader* (Kind of Blue) — 트리오 에너지
- **다이내믹**: mf → f
- **세부**:
  - 앨범에서 가장 밝고 빠름. 너의 향이 절정으로 가득 찬 순간.
  - 피아노·베이스·드럼 세 악기 모두 인터플레이 — 서로 호응
  - 1:30~2:30 클라이맥스 — 트리오 풀 인터플레이 절정
  - 끝부분 8마디부터 드럼 브러시 소멸하며 잔향만 남김 → 4번으로 자연 이동
  - 메이저 → 마이너 전조 = "절정 안에 이미 끝의 예감"

### Side B — 머무름

#### 4. 너의 잔향 (Your Lingering)
- **러닝타임**: 4:30
- **BPM**: 84 / **키**: A minor
- **편성**: 피아노 + 업라이트 베이스 (드럼 완전 퇴장)
- **레퍼런스**: Bill Evans *Peace Piece* 전반부 / Scott LaFaro 베이스라인 해석
- **다이내믹**: mp
- **세부**:
  - 3번 멜로디를 단조로 변주 (장→단 거울)
  - 너는 떠났지만 향은 진하게 남아있음 — 풍성한데 비어있는 모순
  - 업라이트 베이스 라인 중심으로 진행 (3번엔 드럼이 채우던 자리) = "혼자가 된 흔적"

#### 5. 붙잡으려고 (Trying to Hold)
- **러닝타임**: 4:00
- **BPM**: 80 → 76 → 72 (점진적 감속) / **키**: A minor
- **편성**: 피아노 트리오 시작 → 베이스 퇴장 → 피아노 솔로로 축소
- **레퍼런스**: Bill Evans *Alone* (솔로 피아노) — 트리오에서 혼자가 되는 과정
- **다이내믹**: mp → p (점진 감소)
- **세부**:
  - 같은 8마디 프레이즈 → 6마디 → 4마디 → 2마디 (붙잡으려 할수록 작아짐)
  - 박자 미세하게 흔들림 (rubato 강화) = 손에서 빠져나감
  - 마지막 30초 박자 완전히 사라지고 피아노 단음 + 잔향만

#### 6. 미들노트 (Heart Note)
- **러닝타임**: 6:30 (앨범 최장)
- **BPM**: 54 (최저) / **키**: D minor (1번과 같은 키, 다른 정서)
- **편성**: 피아노 + 업라이트 베이스 (arco, long bow) + **남성 보컬리즈 (Chet Baker 스타일)** + 룸톤
- **레퍼런스**: Bill Evans *Peace Piece* + Chet Baker *Almost Blue*
- **다이내믹**: p, 정적
- **세부**:
  - 앨범의 정서적 중심. 가장 느리고 깊음.
  - 업라이트 베이스 arco(활) long tone 위주 — 저역을 가득 채우되 움직이지 않음
  - 피아노는 화음 보조, 거의 움직이지 않음
  - **남성 보컬리즈 두 번째 등장** — 2번과 같은 인물, 같은 톤. 다만 **음역을 한 옥타브 낮춰 베이스 영역(약 G2~D3)** 으로. 더 어둡고 깊은 잔향. 여전히 Chet Baker 방식 — 숨 섞인 흘리기.
  - "가장 오래 남는 향이 가장 슬프다" — 본질에 닿는 트랙
  - 중간 1분 침묵에 가까운 베이스 페달톤만 남음 (앨범 가장 비어있는 순간)

### Side C — 비워짐

#### 7. 휘발 (Evaporation)
- **러닝타임**: 4:00
- **BPM**: 88 / **키**: D minor → C minor
- **편성**: 피아노 트리오 시작 → 드럼 브러시 퇴장 → 업라이트 베이스 퇴장 → 피아노 솔로 + 룸톤
- **레퍼런스**: Miles Davis *Kind of Blue* 인터플레이 → Bill Evans 솔로로 해체
- **다이내믹**: p → pp → ppp
- **세부**:
  - **편곡으로 휘발 표현 (후처리 X)**: 트리오 악기를 실제로 멈추는 방식
  - 빠지는 순서: 드럼 브러시 (첫 32마디 후 퇴장) → 업라이트 베이스 (중반 퇴장) → 피아노 솔로 + 룸톤만
  - 같은 8마디 코드 진행이 4회 반복되며 매번 한 악기씩 사라짐
  - "잡으려 해도 매일 줄어드는 향"의 구체적 사운드 표현 — 곡 자체가 비어가는 과정을 들려줌

#### 8. 또 다른 손 (Another Hand)
- **러닝타임**: 1:45 (앨범 최단)
- **BPM**: 72 (2번과 같음) / **키**: F major (2번과 같음)
- **편성**: 피아노 트리오 + **남성 보컬리즈 (Chet Baker 스타일, 룸마이크 원거리)**
- **레퍼런스**: Bill Evans (모티프 인용 기법) + Chet Baker *Almost Blue*
- **다이내믹**: pp → p
- **세부**:
  - **2번 "시향" 모티프 B(상승 5음)를 인용** — 누군가 또 다가옴
  - 같은 BPM·키지만 톤이 더 어둡고 지침
  - **남성 보컬리즈 마지막 등장** — 2번과 같은 인물·같은 음역대(G3~D4), 같은 Chet Baker 방식. 다만 **마이킹을 멀리(룸 마이크 위주)** 두어 거리감을 표현. 같은 사람의 같은 목소리가 작아진 느낌.
  - 짧게 끝남 = 이번엔 시작도 못 함, 또는 짧은 호기심으로 끝남
  - "이 사람 뭐든 담을 수 있겠다 → 가까이 → 아무것도 없네 → 떠남"의 압축

#### 9. 다시, 무색무취 (Again, Colorless)
- **러닝타임**: 3:30 (룸톤 30초 포함)
- **BPM**: 52~56 (루바토, 1번과 같은 BPM — 체감상 더 느림) / **키**: D minor
- **편성**: 피아노 트리오 인트로 → 드럼 먼저 퇴장 → 베이스 페달톤만 → 피아노 솔로 + 룸톤
- **레퍼런스**: Bill Evans *Peace Piece* (트리오 해석) — 1번 Blue in Green의 거울상
- **다이내믹**: pppp (Track 1보다 한 단계 더 작게)
- **세부**:
  - **1번과 같은 멜로디 모티프 A(D-C-A-F)** — 수미상관 완성
  - 차이점 (1번 대비):
    - 음 사이 침묵: 2~3초 → 3~4초 (1번의 2배)
    - 다이내믹: ppp → pppp (한 단계 더 작게)
    - 베이스 움직임: 4~6박 → 8~10박에 한 번 (절반으로 줄임)
    - 드럼: 마지막 1분 완전 침묵으로 퇴장
    - 마지막 음 페이드아웃 없이 **자연 감쇠 30초** — 마이크가 꺼지지 않고 방을 그대로 녹음
    - 마지막 5초 — 시계 초침 같은 미세한 룸톤만
  - "처음 비어있음(가능성)" → "또 비어버림(체념)"의 거울상

## 사운드 디자인 원칙 (재즈 트리오)

- **트리오 순수주의**: 피아노 + 업라이트 베이스 + 브러시 드럼 세 악기 외 추가 없음. 신스 패드, 현악 4중주 일절 금지.
- **공간이 4번째 악기**: 피아노 노트 사이 침묵 2~4초. 베이스는 박자마다 치지 않음. 드럼은 공기 질감만.
- **보컬리즈 = Chet Baker 방식**: 숨 섞인 흘리는 허밍. 가사 없음. 2·6·8번만 등장. 남성 한 인물, 전 트랙 동일 톤.
- **모달 접근법**: 코드 체인지 최소. 한 모드/페달톤 위에 오래 머물기. Kind of Blue처럼 해소 없는 화성.
- **휘발은 편곡으로**: 트리오 악기를 실제로 멈추는 방식. 드럼 → 베이스 → 피아노 솔로 순 퇴장.
- **트랙 간 단절 금지**: 룸톤이 다음 트랙으로 이어지게 (gapless)
- **후처리 의존 금지**: Suno 생성 단계에서 끝낸다 (Suno 후가공 어려움 고려)

## 멜로디 모티프 매핑
