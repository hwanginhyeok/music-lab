# 무색무취의 빈병

> 인스트루멘털 앨범 / 9 트랙 / 초안 v0.3 (2026-05-04)

## 컨셉

향수병으로서의 자아. 본래 색도 향도 없는 빈 용기. 누군가 들어와야 향이 나지만, 그 향마저 휘발성이라 매일 조금씩 날아가고 결국 다시 무색무취로 돌아간다. 새 사람이 다가올 때마다 같은 사이클이 반복된다.

**가사 없음.** 모든 서사는 사운드·악기·다이내믹·공간감으로만 전달.

**보컬리즈는 남성, 한 명, 전 트랙 일관.** (2·6·8번 3곡에만 등장. 한 인물의 흔적이 사이클 내내 같은 목소리로 남는 효과)

**후처리 최소화 원칙**: Suno 생성 단계에서 작곡·편곡·악기 선택으로 모든 표현을 끝낸다. 별도 후처리(EQ 자동화, 멀티밴드 컴프 등) 의존 금지.

**톤 방향**: 중간 (Max Richter 중심) — Nils Frahm·Joep Beving의 차가운 미니멀과 Hania Rani·Ólafur Arnalds의 따뜻한 서정을 절반씩.

## 핵심 모티프 (사운드 언어)

| 모티프 | 사운드 표현 |
|---|---|
| 빈병 | 공간감, 룸톤, 리버브 테일, 침묵 |
| 향이 든다 | 악기 레이어 추가, 따뜻한 패드, 보컬리즈 |
| 향이 휘발 | 악기 하나씩 빠지기, 노트 수 줄이기, 다이내믹 감소 |
| 다시 빈병 | 솔로 악기 1개 + 룸톤만 남음 |
| 사이클 | 1번 ↔ 9번 멜로디 수미상관, 2번 모티프 → 8번 인용 |

향수의 **탑·미들·베이스 노트** 구조를 트랙 진행에 매핑:
- 탑노트 (1-3): 만남, 화려함, 휘발성 가장 높음
- 미들노트 (4-6): 본질, 가장 오래 머무는 향
- 베이스노트 (7-9): 잔향, 사라짐, 다시 비어감

## 레퍼런스 아티스트 (5)

### Nils Frahm 🇩🇪 — "피아노로 공간을 녹음한다"
펠트 피아노(해머에 펠트를 끼워 둔탁하고 친밀한 음). 마이크를 피아노 안 쇠줄 위에 두고 녹음 → 해머·페달·의자 소리까지 들림. **음표보다 음표 사이의 공기**.
- 대표곡: *Says*, *Felt*, *All Melody*
- 우리: 1번/9번 빈병의 친밀한 마이킹, 7번 공간감 페이드

### Ólafur Arnalds 🇮🇸 — "피아노+현 위에 일렉트로닉을 살짝"
아이슬란드 풍경. 현악 4중주 + 미니멀 일렉트로닉. **안개처럼 깔리는 슬픔**. *Stratus*는 두 대 피아노가 인간+알고리즘으로 동시에 침.
- 대표곡: *Near Light*, *Re:member*, *Ágúst*
- 우리: 3번 풀 어레인지의 균형, 4번 장→단 변주

### Max Richter 🇩🇪/🇬🇧 — "구조의 미학자"
한 모티프를 앨범 내내 다른 편곡으로 변주. *Sleep*(8시간) 같은 모티프 수십 회. **내장된 슬픔(restrained grief)**. 영화 음악 (Arrival, Ad Astra) 작업.
- 대표곡: *On the Nature of Daylight*, *Vladimir's Blues*
- 우리: **수미상관(1↔9) 교과서**, 2→8 모티프 인용, 5번 점진적 축소 반복

### Hania Rani 🇵🇱 — "피아노+자기 목소리를 악기로"
폴란드, 30대 초반. 피아노+신스+본인 보컬리즈("아아", 허밍). 목소리를 가사 아닌 **레이어 악기**로. 따뜻한 미니멀.
- 대표곡: *Esja*, *Eden*, *F Major*, *Hello*
- 우리: 2번/6번 보컬리즈의 모델, "가사 없는 인간성"

### Joep Beving 🇳🇱 — "느림. 침묵. 단순함의 극단"
키 198cm 네덜란드인, 35세 늦게 시작. 솔로 피아노만. 한 마디 4-5음. **음 사이 침묵이 음보다 길다**. Deutsche Grammophon.
- 대표곡: *Sleeping Lotus*, *Ab Ovo*, *Solipsism*
- 우리: 9번 마지막 30초 룸톤·자연 감쇠, "침묵을 견디는 법"

## 트랙리스트 (9트랙)

> BPM·키는 v0.2 1차안. 1번/9번 만든 후 조정 가능.
> 키 동선: D minor(빈병/우울) → F major(따뜻함) → A minor(절정/단조) → C minor(체념) → D minor(회귀)

### Side A — 채워짐

#### 1. 무색무취 (Colorless, Odorless)
- **러닝타임**: 2:30
- **BPM**: 60 / **키**: D minor
- **편성**: 솔로 펠트 피아노 + 룸톤
- **레퍼런스**: Nils Frahm *Says* (질감) + Joep Beving *Ab Ovo* (음 밀도)
- **다이내믹**: ppp~p
- **세부**:
  - 한 마디 3-4음. 노트 사이 침묵 1.5~2초.
  - 페달링 길게 (잔향 = 빈 공간의 부피)
  - 마이킹: 피아노 안쪽 근접 → 해머·펠트 마찰 들리게
  - 멜로디 모티프 A: 4음 하강형 (D-C-A-F) — 9번에서 회귀
  - 인트로 성격, 청자가 "병 안에 들어왔다" 느낌

#### 2. 시향 (First Spray)
- **러닝타임**: 3:30
- **BPM**: 72 / **키**: D minor → F major (전조)
- **편성**: 피아노 + 신스 패드 + **남성 보컬리즈(허밍)**
- **레퍼런스**: Hania Rani *Esja* (보컬리즈 진입) + Ólafur Arnalds *Near Light*
- **다이내믹**: p → mp
- **세부**:
  - 1번 끝 룸톤이 그대로 이어짐 (트랙 단절 없음)
  - 신스 패드 페이드인 8마디 → 호기심 모티프(상승 3도)
  - 보컬리즈 첫 등장 (1:30 지점). **남성, 중음역(테너~바리톤 경계, 약 G3~D4), 가사 없이 "음—" "아—" 한 음 길게**
  - 멜로디 모티프 B: 상승 5음 (F-A-C-E-G) — 8번에서 인용
  - F major 전조 = 누군가 들어옴, 따뜻해짐

#### 3. 탑노트 (Top Note)
- **러닝타임**: 4:00
- **BPM**: 96 / **키**: F major → A minor
- **편성**: 풀 어레인지 — 피아노 + 현악 4중주 + 신스 + 부드러운 드럼(브러시)
- **레퍼런스**: Ólafur Arnalds *Re:member* + Max Richter *Spring 1*
- **다이내믹**: mf → f
- **세부**:
  - 앨범에서 가장 밝고 빠름. 너의 향이 절정으로 가득 찬 순간.
  - 1:30~2:30 클라이맥스 — 전 악기 풀 레이어
  - 끝부분 8마디부터 드럼 빠지며 잔향만 남김 → 4번으로 자연 이동
  - 메이저 → 마이너 전조 = "절정 안에 이미 끝의 예감"

### Side B — 머무름

#### 4. 너의 잔향 (Your Lingering)
- **러닝타임**: 4:30
- **BPM**: 84 / **키**: A minor
- **편성**: 피아노 + 현악 (드럼 OUT)
- **레퍼런스**: Ólafur Arnalds *Ágúst* + Max Richter *On the Nature of Daylight*
- **다이내믹**: mp
- **세부**:
  - 3번 멜로디를 단조로 변주 (장→단 거울)
  - 너는 떠났지만 향은 진하게 남아있음 — 풍성한데 비어있는 모순
  - 첼로 솔로 라인 추가 (3번엔 없던 악기) = "혼자가 된 흔적"

#### 5. 붙잡으려고 (Trying to Hold)
- **러닝타임**: 4:00
- **BPM**: 80 → 76 → 72 (점진적 감속) / **키**: A minor
- **편성**: 피아노 + 현 (스타카토 → 레가토로 무너짐)
- **레퍼런스**: Max Richter *Sleep* 시리즈 (점진 축소 반복)
- **다이내믹**: mp → p (점진 감소)
- **세부**:
  - 같은 8마디 프레이즈 → 6마디 → 4마디 → 2마디 (붙잡으려 할수록 작아짐)
  - 박자 미세하게 흔들림 (rubato 강화) = 손에서 빠져나감
  - 마지막 30초 박자 완전히 사라지고 잔향만

#### 6. 미들노트 (Heart Note)
- **러닝타임**: 6:30 (앨범 최장)
- **BPM**: 54 (최저) / **키**: D minor (1번과 같은 키, 다른 정서)
- **편성**: 첼로 솔로 + 저음 신스 패드 + **남성 보컬리즈** + 룸톤
- **레퍼런스**: Hania Rani *Eden* + Max Richter *Vladimir's Blues* + Joep Beving
- **다이내믹**: p, 정적
- **세부**:
  - 앨범의 정서적 중심. 가장 느리고 깊음.
  - 첼로 long tone 위주, 피아노는 화음 보조
  - **남성 보컬리즈 두 번째 등장** — 2번과 같은 인물, 같은 톤. 다만 **음역을 한 옥타브 낮춰 베이스 영역(약 G2~D3)** 으로. 더 어둡고 깊은 잔향.
  - "가장 오래 남는 향이 가장 슬프다" — 본질에 닿는 트랙
  - 중간 1분 침묵에 가까운 패드만 남음 (앨범 가장 비어있는 순간)

### Side C — 비워짐

#### 7. 휘발 (Evaporation)
- **러닝타임**: 4:00
- **BPM**: 88 / **키**: D minor → C minor
- **편성**: 풀에서 시작 → 점차 비워짐 (악기 자체를 빼는 방식)
- **레퍼런스**: Nils Frahm *All Melody* + Brian Eno *Music for Airports*
- **다이내믹**: p → pp → ppp
- **세부**:
  - **편곡으로 휘발 표현 (후처리 X)**: 악기를 한 트랙씩 연주를 멈추는 방식
  - 빠지는 순서 (높은 음역 악기부터): 글로켄슈필/벨 → 바이올린 하이 → 비올라 → 피아노 우손 → 첼로 → 신스 패드 → 마지막 베이스 패드만
  - 같은 8마디 코드 진행이 4회 반복되며 매번 한 악기씩 사라짐 (사라졌다 돌아오는 느낌은 패드 강약으로만)
  - "잡으려 해도 매일 줄어드는 향"의 구체적 사운드 표현 — 곡 자체가 비어가는 과정을 들려줌

#### 8. 또 다른 손 (Another Hand)
- **러닝타임**: 1:45 (앨범 최단)
- **BPM**: 72 (2번과 같음) / **키**: F major (2번과 같음)
- **편성**: 피아노 + 신스 패드 + **남성 보컬리즈**
- **레퍼런스**: Max Richter (모티프 인용 기법) + Ólafur Arnalds
- **다이내믹**: pp → p
- **세부**:
  - **2번 "시향" 모티프 B(상승 5음)를 인용** — 누군가 또 다가옴
  - 같은 BPM·키지만 톤이 더 어둡고 지침
  - **남성 보컬리즈 마지막 등장** — 2번과 같은 인물·같은 음역대(G3~D4)지만, **마이킹을 멀리(룸 마이크 위주)** 두어 거리감을 표현. 같은 사람의 같은 목소리가 작아진 느낌.
  - 짧게 끝남 = 이번엔 시작도 못 함, 또는 짧은 호기심으로 끝남
  - "이 사람 뭐든 담을 수 있겠다 → 가까이 → 아무것도 없네 → 떠남"의 압축

## 사운드 디자인 원칙

- **ASMR급 근접 마이킹**: 청자가 병 안에 들어와 있는 듯한 친밀감
- **공간감 = 비어있음**: 리버브를 정서가 아니라 "용기의 부피"로 사용
- **악기 최소화**: 한 트랙에 5개 이상 안 쓰기. 빈자리가 향이 빠진 자리
- **보컬리즈**: 인간성이 필요한 순간만(2·6·8). **남성 보컬, 한 인물, 전 트랙 일관**. 절대 가사로 가지 않음
- **휘발은 편곡으로**: 향의 휘발 = 악기를 빼는 방식으로 표현 (후처리 EQ 자동화 X)
- **트랙 간 단절 금지**: 룸톤이 다음 트랙으로 이어지게 (gapless)
- **후처리 의존 금지**: Suno 생성 단계에서 끝낸다 (Suno 후가공 어려움 고려)

## 멜로디 모티프 매핑

- **모티프 A** (D-C-A-F, 4음 하강): 1번 도입 → 9번 회귀
