# Suno 고급 리서치 프롬프트

Suno AI로 더 정교하고 퀄리티 높은 노래를 만드는 방법을 심층 리서치해줘. 웹 검색(/browse) 적극 활용해서 최신 2025년 정보까지 포함할 것.

## 조사할 항목

### 1. 프롬프트 엔지니어링 고급 기법
- Suno의 Style Prompt 작성 요령 (태그 vs 자연어, 길이, 순서)
- 메타태그 활용 (BPM, Key, 장르 세분화, vocal type 지정)
- negative prompt 활용법
- instrumental vs vocal 트랙 제어

### 2. Suno v4/v3.5 최신 기능 활용
- Audio Input (기존 음원 기반 생성) 활용법
- Extend 기능으로 곡 구조 설계 (verse-chorus-bridge 제어)
- Person 봇/커스텀 보이스 기능
- Reuse/Remix 전략

### 3. 곡 구조 설계 (Structure Tags)
- [Intro], [Verse], [Pre-Chorus], [Chorus], [Bridge], [Outro], [Break] 태그 사용법
- 반복 vs 변형 전략
- 곡 길이 제어 (2분 vs 4분)

### 4. 작사(Lyrics) 최적화
- Suno가 잘 부르는 가사 패턴 (음절 수, 라임, 발음)
- 한국어 가사 작성 시 주의점
- 영어+한국어 혼합 가사 처리
- Ad-lib, 화음, 백보컬 태그 삽입

### 5. 후처리 & 업스케일링
- Suno 출력물 품질 한계 및 보완 방법
- DAW 후처리 워크플로우
- 스템 분리 후 리믹싱

### 6. 프로덕션 파이프라인
- 반복 생성 → 선별 → 조합 사이클
- 버전 관리 (어떤 프롬프트가 어떤 결과를 냈는지)
- 자동화 가능한 부분 (API 활용)
