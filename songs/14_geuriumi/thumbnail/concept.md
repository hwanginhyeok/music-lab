# 썸네일 컨셉 — 왜 그리 울고만 있어요? 그리움만 쌓이게

## 핵심 컨셉
'그리움이 쌓인다'의 시각화. 작은 아이가 흰 배경에서 조용히 울고 있고,
1초 뒤 머리에 눈이 살짝 쌓여 있다. 자세는 그대로, 시간만 흘렀다는 단서.
'그리(부사) 우는 거 그만해 / 그리움만 쌓이니깐' 의 메타포 그대로.

## 두 장면 (시간차)

### Image A (t=0) — 막 내리기 시작
- 흰 배경 (눈/안개 같은 무한 공간)
- 작은 아이 한 명, 화면 중앙 또는 약간 좌측, 정면 또는 4분의 3 각도
- 눈물이 한두 방울 뺨에 흘러내리는 중
- 눈송이가 화면 안에 흩날림 (막 내리기 시작)
- 머리 위 눈은 거의 없음 (한두 송이 깃털처럼)
- 표정: 조용한 슬픔, 무력함, 절제

### Image B (t=+1s) — 1초 후
- 같은 배경, 같은 자세, 같은 표정
- 단 머리 위에 눈이 살짝 쌓여있음 (얇은 베일처럼)
- 어깨에도 눈송이 몇 개
- 눈물은 그대로 또는 한 방울 더
- 자세 변화 없음 — 시간만 흘렀다는 시각적 단서

## 톤 / 스타일
- 일본 수채화 + 한국 동화책 일러스트
- 또는 Studio Ghibli 식 부드러운 라인
- 또는 Jon Klassen / Beatrice Alemagna식 미니멀 정서 일러스트
- 색감: 99% 무채색 (white, light grey, soft black)
- 단 눈물 한 방울에만 살짝 차가운 푸른 빛 (cool blue tint)
- 선: 가늘고 떨림 있는 손그림 라인 (펜/연필)

## 화면 비율
- 16:9 (YouTube 썸네일/영상 표지)
- 1:1 별도 받으면 Spotify/Instagram 호환

## 텍스트
- 이미지 안에 곡 제목 텍스트 X (재즈 앨범 표지처럼 깔끔)

## 활용
- Image A → 영상 인트로용
- Image B → YouTube 썸네일 (시간 흐른 모습이 더 정서적)
- 또는 영상 마지막에 페이드로 A→B 변화 (1초 차이 시각 효과)
