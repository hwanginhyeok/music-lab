## 정서

졸업. **1번 색스 모티프 A를 D major로 회귀** — 수미상관 완성. 비움이 시작이었고, 비움이 끝이다.

**앨범에서 처음으로 화자가 입을 여는 트랙** — 1-8번 인스트루멘털을 거쳐 9번에서 비로소 영어 가사로 자기를 정리. 한국 아저씨 정서 + 영어 보컬의 미스매치가 거리감(자기 객관화)을 만들어줌. 같은 화자가 자기 자신을 영어로 보내주는 작별.

## 멜로디 모티프

- **모티프 A 회귀** — 1번 D minor에서 9번 D major로. 같은 멜로디, 다른 정서
- **모티프 C 변형** — 6번 색스 외침 → 9번 보컬 멜로디 베이스로

## 보컬 (영어, 굵은 남자 바리톤)

**톤 레퍼런스: Gregory Porter / Kurt Elling / Mark Murphy.** 깊고 따뜻한 바리톤. 컨템포러리 재즈 보컬 스토리텔링. "가수가 직접 불러주는 듯한" 자연스러움이 목표.

**왜 영어 남자 바리톤인가:**
- 5-14 트랙 06 한국어 여성 보컬과 명확히 차별화 (채널 시리즈 안에서)
- 한국 아저씨 정서 + 영어 보컬의 미스매치 = 9번이 가지는 거리감(자기 객관화)
- 굵은 바리톤 = Late-night smoky jazz의 클래식 결 (1950-60s East Coast 재즈 보컬 전통)
- 색스(테너 ~ 바리톤 영역)와 보컬 음역대 매칭 — 한 사람의 두 목소리처럼 들림

**핵심 라인 후보 (lyricist 굳힐 영역):**

영어 다중 의미 트릭 후보:
- **"I'm done"** — 끝났다 / 망가졌다 / 해방됐다
- **"Sober"** — 술 깼다 / 진지해졌다 / 깨달았다
- **"The bottle"** — 술병 / 메시지 / 갇힘

샘플 라인 (방향만, 최종은 lyricist):
- "The bottle is empty / and so am I / empty of you"
- "I drank you out / I drank myself well"

- "I'm done with the bottle / I'm done with you / I'm done"

직접 "I miss you / I love you" 회피. 사물(bottle)·행동(drink/done)·상태(empty/sober/healed)로 우회.
