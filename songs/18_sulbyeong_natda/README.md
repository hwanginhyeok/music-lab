## 핵심 구조 요약

**3·3·3 동음 역설:**
- Part I (1-3) 빈병 = 다 마셨다 = 이별 비움 (마음 아픔)
- Part II (4-6) 몸병 = 과음 = 신체화
- Part III (7-9) 회복 = 병이 나았다 = 졸업

**수미상관:**
- 모티프 A: 1번 색스 도입 4음 → 9번 회귀
- 모티프 B: 3번 색스 페이드 → 7번 회복 시 인용
- 모티프 C: 6번 색스 외침 → 9번 보컬 멜로디 베이스

**장르 일관:** Late-night smoky jazz, 색소폰 메인 보이스 9트랙 내내. 정서 변화는 편성·색스 톤·BPM·키·다이나믹으로만.
