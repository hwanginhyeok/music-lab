# 봄이라고 부를게 — 곡 컨셉

## 기본 정보
- **작업명**: 봄이라고 부를게
- **장르**: 미디엄 발라드 / 어쿠스틱 팝
- **BPM**: 85~95
- **키**: C major → A minor (1절 밝음 → 2절 전환)
- **참조 스타일**: 김이나 작사법
- **버전**: v1.3

## 핵심 컨셉
겨울에 헤어진 사람을 봄에 그리워하는 이야기.
1절은 "봄이 왔으니까 설레는 거야"라고 자기합리화하다가,
2절에서 "사실 네가 보고 싶어서 전화한 거야"로 진심이 드러남.
Final에서 결국 벚꽃 핑계로 전화를 건다.

## 언어유희 맵
```
1절: call = 부르다(이름)  /  봄 = spring(계절)
     "너를 봄이라고 부를게"
     → 이별한 사람을 계절로 부르겠다는 우회

2절: call = 전화하다      /  봄 = 보고 싶다(see)
     "네가 보고 싶어서 전화했어"
     → 포장 벗기고 직진. 핑계 그만.

Final: 두 의미 겹침 → "같은 말인데... 이상하다"
       "오늘 벚꽃이 피었더라, 그래서 전화해봤어..."
       → 결국 다시 핑계로 돌아가지만, 듣는 사람은 진심을 안다
```

## 화자 캐릭터
- 20대 후반~30대 초반
- 이별 후 몇 달 지남 (겨울 → 봄)
- 감정을 직접 말하지 않고 계절/날씨 핑계를 댐
- 자기 감정을 알면서도 인정하지 않으려 함
- 결국 전화를 거는 순간 무너짐
- 하지만 전화 받으면 또 날씨 얘기로 시작할 사람

## 악기 구성 (제안)
```
Verse 1:  어쿠스틱 기타 + 피아노 (가볍게)
Chorus 1: 현악기 진입 + 드럼 라이트
Verse 2:  피아노 중심 (무게감, 기타 빠짐)
Chorus 2: 풀 밴드 — 드럼 + 베이스 + 현악기
Bridge:   피아노 단독 + 전화 효과음 (투르르)
Final:    피아노만 남음 → 점점 사라짐
```

## 코드 진행 (제안)
```
Verse:    C → G/B → Am → F
          (I → V/VII → vi → IV, 표준 팝 진행의 변형)

Chorus:   F → G → Am → C/E
          (서브도미넌트 시작으로 설렘 → 해결)

Bridge:   Am → F → C → G
          (마이너 시작으로 긴장감)

Final:    C → Am → F → ...
          (F에서 해결 안 하고 끊김 — 말하다 끊기는 가사와 매칭)
```

## 레퍼런스 곡 후보
- 아이유 「밤편지」 — 어쿠스틱 + 담담한 고백 톤
- 성시경 「거리에서」 — 미디엄 발라드 편곡감
- 볼빨간사춘기 「여행」 — 봄 느낌의 산뜻한 사운드
