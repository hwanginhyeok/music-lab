# 봄이라고 부를게 — 곡 컨셉

## 기본 정보
- **작업명**: 봄이라고 부를게
- **장르**: 미디엄 발라드 / 어쿠스틱 팝
- **BPM**: 85~95 (봄의 산뜻함 + 그리움의 여유)
- **키**: C major → A minor (1절 밝음 → 2절 전환)
- **참조 스타일**: 김이나 작사법 (캐릭터 설정, 일상 은유, 언어유희)
- **버전**: v1 (초안)

## 핵심 컨셉
겨울에 헤어진 사람을 봄에 그리워하는 이야기.
1절은 "봄이 왔으니까 설레는 거야"라고 자기합리화하다가,
2절에서 "사실 네가 보고 싶어서 전화한 거야"로 진심이 드러남.

## 언어유희 맵
```
1절: call = 부르다(이름)  /  봄 = spring(계절)
     "너를 봄이라고 부를게"

2절: call = 전화하다      /  봄 = 보다(see)
     "너를 보려고 불러볼게"
     "I wanna call you to see you"

Final: 두 의미 겹침 → "같은 말인데 다른 말"
```

## 화자 캐릭터
- 20대 후반~30대 초반
- 이별 후 몇 달 지남 (겨울 → 봄)
- 감정을 직접 말하지 않고 계절/날씨 핑계를 댐
- 자기 감정을 알면서도 인정하지 않으려 함
- 결국 전화를 거는 순간 무너짐
