# 봄이라고 부를게 — v2 (수정 검토본)

> 작사 컨셉: 김이나 스타일 — 일상적 언어, 감각적 은유, 반전의 언어유희
> 핵심 장치: 봄(spring) / 봄(see) + call(부르다) / call(전화하다)
> 구조: 1절(봄의 그리움, 산뜻) → 싸비(너를 봄이라고 부를게) → 2절(겨울 이별 고백) → 싸비(너를 보려고 전화할게)

> 🔴 빨간색 = 음절 수 불일치 심각 (±4 이상), 수정 필요
> 🟡 노란색 = 음절 수 경미 불일치 (±2~3), 검토 필요

## 작사 노트

**언어유희 구조:**
- 1절 싸비: 봄 = spring (계절) / 부르다 = call (이름 부르기) → "너를 봄이라고 부를게"
- 2절 싸비: 봄 = 보고 싶다(see) / call = 전화했어 → "네가 보고 싶어서 전화했어"
  - 1절의 우회적 표현이 벗겨지고 진심이 터지는 순간
  - "I just called you, wanna see you" — called(전화했어) + see(보고싶어) = 봄의 이중 의미
- Final: 전화가 연결된 것처럼 말하다가 끊김. "나 오늘 벚꽃 봤어, 그래서..."
  — 그래서 뭐? 보고 싶었다고? 전화했다고? 말 못하고 끊긴다.
  — "봄은 매년 오잖아, 근데 너는" — 너는 안 와. 라고 말 안 하고 끊는 게 잔인함의 핵심.

**김이나 스타일 반영:**
- "벚꽃 같다고 하면 너무 뻔한 거잖아" — 뻔한 비유를 의식적으로 피하는 자의식 (김이나 특유의 메타 화법)
- "커피잔 위에 김이 피는 것처럼" — 일상 사물로 추상적 감정을 감각화
- "택시 두 대가 반대로 꺾어진 골목" — 구체적 장면으로 이별을 보여주기 (말하지 않고 보여주기)
- "투르르 투르르" — 전화 벨소리 의성어로 브릿지 전환
- 전체적으로 과잉 감정 없이 담담한 화자가 상황을 묘사하고, 감정은 독자가 느끼게 하는 구조
