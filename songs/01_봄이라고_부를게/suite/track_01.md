# Track 01 — 창문 여는 순간

> Suite의 첫 숨. 아침에 창문을 열면 들어오는 3월 공기 — 차갑지도 뜨겁지도 않은, 그냥 새로운 공기.

---

## 감정

나른함. 아직 설렘이라고 부르기엔 이른, 그냥 "오늘 뭔가 좋을 것 같은" 예감.

## Style of Music
```
solo piano, jazz, rubato, gentle, warm, contemplative, spring mood, morning light, 75bpm
```

## Lyrics (Suno 형식)

```
[Instrumental Intro]
[Solo Piano, Rubato]

[Piano Solo]
[Gentle, Free Tempo]

[Build]
[Slightly Warmer]

[Outro]
[Fade Into Next]
```

## 연출 노트

- 루바토(자유 박자)로 시작. 메트로놈 없이 피아니스트가 혼자 치는 느낌.
- 중반부터 살짝 리듬감이 생기면서 Track 2의 보사노바로 이어질 준비.
- Rhodes가 아닌 **어쿠스틱 피아노**. Suite에서 순수 피아노는 여기와 Track 5뿐.
- 키: Eb Major. Suite 전체의 홈 키를 여기서 제시.
