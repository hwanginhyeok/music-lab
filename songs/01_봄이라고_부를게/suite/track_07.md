# Track 07 — 눈물 마른 뒤의

> 폭발 직후. 울음이 멈추긴 했는데 몸이 아직 떨리고 있다. 숨이 고르지 않고, 한숨이 나오고, 멍하게 앉아있는.

---

## 장르: Jazz Ballad (잔열)

재즈에서 가장 느리고 감정적인 스타일. 템포 60~80 BPM.
핵심: 음 하나하나가 길게 울린다. 빠른 재즈가 "얼마나 많은 음을 치나"라면, 발라드는 "음 하나를 얼마나 아름답게 치나".
팝 발라드와 다른 점: 화성이 훨씬 복잡하고, 악기 간 즉흥적 호흡이 있다.

### 이 트랙에서 쓴 태그
| 태그 | 들을 때 포인트 |
|------|---------------|
| `jazz ballad` | 느린 재즈. 음이 적고 하나하나가 무겁다 |
| `Rhodes electric piano` | 따뜻하고 둥근 전기 피아노 |
| `tenor saxophone` | Track 6에서는 울부짖었는데 여기서는 떨리면서 가늘어짐. 같은 악기가 지쳐가는 과정 |
| `vulnerable` | 방어 없이 열린 상태. 감정이 다 드러난 뒤의 벌거벗은 느낌 |
| `trembling` | 떨림. 울음 직후 몸이 아직 가라앉지 않은 |
| `exhale` | 한숨. 길게 내쉬는 호흡. 악기 프레이즈가 내려가는 방향 |
| `85bpm` | Track 6(100)에서 급락하지 않고 서서히 내려온 속도 |

---

## 감정

잔열. 차분해진 게 아니라 **아직 뜨거운 채로 멍한** 상태. "하..." 하고 한숨 쉬는데 눈물 자국이 아직 안 말랐다.

## Style of Music
```
jazz ballad, Rhodes electric piano, tenor saxophone, upright bass, brush drums, vulnerable, trembling, exhale, warm, spring mood, 85bpm
```

## 연출 노트

- **색소폰으로 시작한다**. Track 6의 마지막 색소폰이 끊기지 않고 Track 7로 이어지는 느낌. 처음엔 아직 거칠고 떨리다가 점점 가늘어진다.
- Rhodes가 뒤늦게 들어오면서 색소폰을 받아준다. "괜찮아" 하듯이.
- 색소폰이 중반에 빠지고 Rhodes만 남는다 — 감정이 서서히 내려가는 과정.
- 브러시 드럼은 있는지 없는지 모를 정도로. 맥박만 유지.
- **핵심: 처음부터 조용한 게 아니다.** Track 6의 열기를 안고 시작해서 4분에 걸쳐 식는다.
- Track 8로 넘어갈 때 Rhodes가 마지막 음을 길게 울리며 끝.

## Revision Log

### rev 1 → rev 2 (2026-04-02)

**변경 사유**: Track 6에서 엄청 울었는데 Track 7이 갑자기 차분해짐. 감정이 아직 남아있는 상태("하..." 한숨, 몸이 떨리는)인데 `aftermath`+`gentle`+`72bpm`이 처음부터 다 식은 상태로 만들어버림.

**핵심 변경**:

| 항목 | rev 1 | rev 2 |
|------|-------|-------|
| 감정 | 카타르시스 후 고요, 개운함 | 잔열, 아직 뜨거운 채로 멍한 |
| 템포 | 72 BPM (급냉) | 85 BPM (서서히 내려옴) |
| 시작 | Rhodes 혼자 조용히 | 색소폰이 Track 6에서 이어지며 시작 |
| 제거 태그 | — | `aftermath` (처음부터 끝난 느낌 유발), `gentle` (너무 차분) |
| 추가 태그 | — | `trembling` (떨림), `exhale` (한숨) |
| 구조 | Rhodes 시작 → 색소폰 진입 | 색소폰 시작 → Rhodes가 받아줌 → 색소폰 퇴장 |

**의도**: Track 6의 열기를 안고 시작해서 **4분에 걸쳐 천천히 식는다**. 처음부터 차분한 게 아니라 뜨거운 것이 식어가는 과정 자체가 이 트랙.
