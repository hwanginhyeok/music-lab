# Track 07 — 눈물 마른 뒤의

> 폭발 직후의 잔향. 울고 난 뒤에 오는 이상한 평온함. Rhodes가 천천히 흐른다.

---

## 장르: Jazz Ballad

재즈에서 가장 느리고 감정적인 스타일. 템포 60~80 BPM.
핵심: 음 하나하나가 길게 울린다. 빠른 재즈가 "얼마나 많은 음을 치나"라면, 발라드는 "음 하나를 얼마나 아름답게 치나".
팝 발라드와 다른 점: 화성이 훨씬 복잡하고, 악기 간 즉흥적 호흡이 있다.

### 이 트랙에서 쓴 태그
| 태그 | 들을 때 포인트 |
|------|---------------|
| `jazz ballad` | 느린 재즈. 음이 적고 하나하나가 무겁다 |
| `Rhodes electric piano` | 따뜻하고 둥근 전기 피아노. Track 6의 어쿠스틱 피아노와 톤 차이에 주목 |
| `tenor saxophone` | Track 6에서는 거칠게 울부짖었는데 여기서는 속삭임. 같은 악기 다른 얼굴 |
| `gentle` | 부드러움. Track 6의 `raw`/`gritty`와 정반대 |
| `vulnerable` | 약한 모습을 드러내는. 방어 없이 열린 상태 |
| `aftermath` | "이후". 큰 일이 지나간 뒤의 고요함 |
| `72bpm` | 심장박동보다 느린. 시간이 천천히 가는 느낌 |

---

## 감정

카타르시스 이후의 고요. 슬프지만 개운한. 울어서 오히려 가벼워진 마음.

## Style of Music
```
jazz ballad, Rhodes electric piano, tenor saxophone, upright bass, brush drums, gentle, vulnerable, aftermath, warm, spring mood, 72bpm
```

## Lyrics (Suno 형식)

```

```

## 연출 노트

- Track 6의 거친 에너지와 대비되는 **부드러움**. 같은 악기(Rhodes, 색소폰)인데 톤이 완전히 다르다.
- 색소폰이 Track 6에서는 울부짖었다면, 여기서는 속삭인다. 같은 악기의 두 얼굴.
- 브러시 드럼이 돌아온다. Track 6에서 스틱으로 세게 쳤던 드럼이 다시 브러시로.
- 템포 72 BPM — Suite에서 두 번째로 느림. 호흡을 가다듬는 시간.
- Track 8로 넘어갈 때 템포가 자연스럽게 올라가야 한다. 아웃트로에서 살짝 빨라지는 느낌.
